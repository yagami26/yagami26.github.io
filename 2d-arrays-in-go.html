Creating 2-D arrays in golang

Tue 21 May 2024 Technology by khirod
Tags: golang, syntax

I was doing leetcode problems because being prepared doesn't harm anyone, haha, and I'm also trying to learn golang in parallel. Going through the basic syntax currently and a lot of time scenarios arise where you have to initialise a 2-D array eg- graphs, or for dynamic programming problems. Here we use built-in make function to create a slice in golang. So this implementation is a slice of slices. I read there is other way of creating 2D arrays as well but I'll explore that later as this seems simpler to me. 

package main

import "fmt"

func main() {
    const columns = 5;
    const rows = 6;

    var multiDimensionalArray [][]int = make([][]int, rows);
    for i := 0; i < rows; i++ {
        multiDimensionalArray[i] = make([]int, columns);
    }

    // Accessing rows and columns
    for i := 0; i < rows; i++ {
        for j := 0; j < columns; j++ {
            multiDimensionalArray[i][j] = 1;
        }
    }

    fmt.Println(multiDimensionalArray);

}
