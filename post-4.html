How to copy from vim to global clipboard on linux

May 24, 2024

Having installed arch linux again, I am going back to vim style of editing the files. I use neovim specifically and recently I struggled with how to yank lines from your editor to a global clipboard. This is needed at times when you want to copy something from your editor like code for eg to a clipboard and paste it in browser.

I didn't knew that anything which is yanked or deleted in vim is stored into a register. To see the current content of all the registers, run :reg in command mode.

How to copy

  • First you have to go into visual mode in vim using Shift + V then you can select the lines you want to copy, if it is the entire thing to end of file then you can also press G.

  • Once the lines are selected the next step is to yank them into a system clipboard register. In case of system clipboard in linux the register is +. So to copy into clipboard we simply do it using "+y to yank the selected lines in visual mode
